# localrank

* **Version:** 0.1-0
* **URL**: https://github.com/RBigData/localrank
* **License:** [BSD 2-Clause](http://opensource.org/licenses/BSD-2-Clause)
* **Author:** Drew Schmidt


A backport of the pbdMPI function `comm.localrank()`. Meant for users on HPC resources who can't update to the latest pbdMPI, which requires a recent version of R.

In 5 years when your HPC center updates R to 3.5.x, this package will be useless.


## Installation

The development version is maintained on GitHub:

```r
remotes::install_github("RBigData/localrank")
```
